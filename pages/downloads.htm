EZ-Zip

Basic tool written in bash to combine all linux archive tools into one place. This was written in hopes to make it easier to manage archives. This tool is compatble with .zip files, tar files, .gz files, and .tar.gz files.

Premium Bowling Texture Index

This is a very simple site that hosts bowling ball textures for the game premium bowling. It has a search bar to simplify the finding of textures and it is regularly updated with textures of recently released bowling balls. Feel free to request any bowling balls you would like to see added once I get my contact page up.
